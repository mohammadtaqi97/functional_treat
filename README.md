# functional_treat
Data Analyser and Transformer Program

Description

This is a Python menu-driven project that allows users to analyse and transform numerical data in different ways. Users can enter dataset values and perform operations like summary generation, factorial/fibonacci calculation, filtering, sorting, and statistics display.

Features

1. Input Data
2. Display Data Summary
3. Calculate Factorial or Fibonacci
4. Filter Data by Threshold
5. Sort Data
6. Display Dataset Statistics
7. Exit Program

Technologies Used

- Python
- Lists
- Loops
- Conditional Statements
- Functions like min(), max(), sum(), len()

How to Run

1. Open Python IDLE / VS Code
2. Save the file as "data_analyser.py"
3. Run:
   python data_analyser.py
